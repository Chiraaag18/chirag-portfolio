# Chirag Portfolio

A static personal portfolio built from the resume content in `Chirag's Resume 17-04.pdf`.

## Local preview

Because this is a plain static site, you can preview it with any simple static file server. One easy option is:

```powershell
npx serve .
```

Then open the local URL shown in the terminal.

## Files

- `index.html` contains the portfolio structure and content
- `styles.css` contains the visual design and responsive layout
- `script.js` adds reveal animations and scroll-to-top behavior

## Free hosting options

### Vercel

1. Push this repo to GitHub.
2. Import the repo into [Vercel](https://vercel.com/).
3. Deploy as a static site with the default settings.
4. You will get a free `*.vercel.app` domain.

### Netlify

1. Push this repo to GitHub.
2. Import the repo into [Netlify](https://www.netlify.com/).
3. Set the publish directory to the repo root.
4. You will get a free `*.netlify.app` domain.

### GitHub Pages

1. Push this repo to GitHub.
2. In repo settings, enable GitHub Pages from the main branch root.
3. You will get a free `username.github.io/repository-name` URL.
